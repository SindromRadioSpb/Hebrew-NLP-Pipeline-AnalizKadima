# Методика: he_11 — Морфология: глаголы и согласование

## Процесс 1: Verb Form Recognition

### Вход
Глагольные формы в прошедшем и настоящем времени.

### Правила
1. **Прошедшее время (past)**:
   - העובד בדק (masc.sg) / העובדים בדקו (masc.pl) / המעצבת תכננה (fem.sg) / המתכננים תכננו (masc.pl).
   - Различение по числу и роду.
2. **Настоящее время (present participle)**:
   - העובד בודק (masc.sg) / העובדים בודקים (masc.pl) / המעצבת מתכננת (fem.sg) / המתכננים מתכננים (masc.pl).
   - Participle согласуется с подлежащим.

### Lemma
- Все формы → инфинитив: בדק, תכנן, מדד.

### Acceptance criterion
- POS = VERB: exact.
- Lemma = инфинитив: exact (если словарь поддерживает).

## Процесс 2: Gender/Number Agreement

### Вход
Пары существительное + прилагательное.

### Правила
1. **Муж.ед.**: חומר עמיד, פלדה → (жен.р.)
2. **Жен.ед.**: מתכת חזקה, פלדה מוקשחת.
3. **Муж.мн.**:חומרים עמידים.
4. **Жен.мн.**: מתכות חזקות, סגסוגות מוקשחות, טמפרטורות גבוהות.

### Acceptance criterion
- Согласование проверяется через пары: (noun_gender, noun_number) → (adj_gender, adj_number).
- EXACT для предопределённых пар.
