# Соглашения токенизации и лемматизации

## Токенизация

### Базовое правило
Разделитель — **пробел** (` `). Каждый фрагмент между пробелами = один токен.

### Пунктуация
Знаки препинания, приклеенные к слову — **часть токена**:
- `פלדה.` = 1 токен (не 2)
- `(חומר)` = 1 токен (не 3)
- `סגסוגת;` = 1 токен

### Maqaf (־)
Maqaf (еврейский дефис) — **разделитель или часть слова**?
- В данных: `אל-חלד`, `חוזק-משקל`
- Соглашение: maqaf считается **одним токеном** (как дефис в compound)

### Числа
- `520` — 1 токен
- `3.14` — 1 токен (десятичная точка — часть числа)
- `18%` — 1 токен (процент приклеен)
- `5kg` — 1 токен (единица приклеена)
- אלף חמש מאות — 3 токена (3 слова)

### Двойные пробелы
Двойные пробелы НЕ создают пустых токенов. `"פלדה  ברזל"` = 2 токена.

## Lemmatization

### Общее правило
Лемма = словарная форма слова.

### Существительные
- Единственное число, мужской род (если есть)
- `מעלות` → `מעלה`
- `חומרים` → `חומר`
- `סגסוגות` → `סגסוגת`

### Прилагательные
- Единственное число, мужской род
- `גבוהה` → `גבוה`
- `חזקות` → `חזק`
- `מחמירים` → `מחמיר`

### Глаголы
- Инфинитив в базовой форме
-בניין зависит от pipeline (не проверяется как exact)
- `משמשת` → `שימש` (или `שמש`, зависит от словаря)
- `נבדק` → `נבדק` (nif'al = same as base)

### Местоимения
- Базовая форма: `הוא`, `הם`
- `היא` → `הוא`
- `שניהם` → `שניהם`

### Предлоги с артиклем
- `בטמפרטורה` → lemma: `טמפרטורה` (без ב)
- `לברזל` → lemma: `ברזל` (без ל)
- `במעבדה` → lemma: `מעבדה` (без ב)

## DET detachment

### Когда отделять
Определённый артикль **ה** отделяется если:
1. Слово начинается с ה
2. После удаления ה остаётся валидное слово
3. ה — это НЕ часть корня

Примеры:
- `הפלדה` → DET+פלדה ✅
- `הברזל` → DET+ברזל ✅
- `החומר` → DET+חומר ✅
- `הנוזלי` → DET+נוזלי ✅

### Когда НЕ отделять
- `היתוך` (плавление) — ה часть корня ❌
- `הרכב` (состав) — ה часть корня ❌
- `הוא` (он) — местоимение ❌
- `הוספת` (добавление) — ה часть глагольной формы ❌

### Пограничные случаи
| Слово | Решение | Причина |
|-------|---------|---------|
| היתוך | Не отделять | Корень ית"ך, ה не артикль |
| הרכב | Не отделять | Существительное הרכב |
| הוסיף | Не отделять | Глагольная форма |
| הסגסוגת | Отделять | ה = определённый артикль |
| האיכות | Отделять | ה = определённый артикль |

## Подсчёт предложений

### Разделитель
Точка `.` после слова = boundary предложения.

### Исключения
- Аббревиатуры: `פרופ׳`, `ד״ר` — НЕ split
- Десятичные: `3.14` — НЕ split
- Многоточие: `...` — зависит от pipeline (отмечать в review)

### Граничные случаи
- Текст без конечной точки: считается как 1 предложение
- Пустой файл: 0 предложений
- Только точка: 0 предложений (или 1, зависит от pipeline)
