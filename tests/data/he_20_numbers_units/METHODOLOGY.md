# Методика: he_20 — Числа и единицы

## Процесс: Number and Unit Processing

### Вход
Числовые токены (еврейские словесные, арабские цифровые, проценты).

### Правила
1. **Еврейские числительные**: אלף חמש מאות = 3 токена (3 слова).
2. **Арабские числа**: 520 = 1 токен.
3. **Десятичные**: 3.5, 0.3 = 1 токен.
4. **Проценты**: 18% = 1 токен (приклеенный символ).
5. **Число + unit без пробела**: 5kg = 1 токен.
6. **Число + unit с пробелом**: 520 מגהפסקל = 2 токена.
7. **Префикс + число**: ב-15 = 1 токен (дефис приклеен).

### Acceptance criterion
- Token count: exact.
- Number parsing: manual_review (pipeline-dependent).
