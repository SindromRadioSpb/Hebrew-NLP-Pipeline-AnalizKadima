# Методика: he_13 — Canonicalization

## Процесс: Surface-to-Canonical Mapping

### Вход
Поверхностные формы слов в разных грамматических формах.

### Правила
1. **DET removal**: הניקל → ניקל, הסגסוגת → סגסוגת.
2. **Number normalization**: סגסוגות → סגסוגת (ед.ч.).
3. **Construct normalization**: טמפרטורת → טמפרטורה.
4. **Adj agreement**: גבוהות → גבוה (муж.ед.).
5. **Combined**: הטמפרטורותהגבוהות → טמפרטורה + גבוה.

### Canonical form
Одно уникальное представление для каждого понятия, независимо от поверхностной формы.

### Acceptance criterion
- All variant surfaces map to same canonical: exact.
- Canonical is deterministic (same input → same output).
