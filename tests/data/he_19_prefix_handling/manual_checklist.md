Проверяет обработку еврейских префиксов ב/ל/כ/מ/וש.

## Рекомендуемый профиль

any

### Шаг 3: Prefix stripping
- [ ] בפלדה → ב + פלדה → lemma: פלדה
- [ ] לברזל → ל + ברזל → lemma: ברזל
- [ ] באלומיניום → ב + אלומיניום → lemma: אלומיניום
- [ ] וובפלדה → ו + ב + פלדה → lemma: פלדה
- [ ] אבלברזל → א + ב + ל + ברזל? או אבל + ברזל?

### Шаг 4: Key checks
- Single prefix should be stripped correctly
- Combined prefixes (ו+ב, ו+ל) should be handled
- Ambiguous combinations (אבלברזל) — document behavior
