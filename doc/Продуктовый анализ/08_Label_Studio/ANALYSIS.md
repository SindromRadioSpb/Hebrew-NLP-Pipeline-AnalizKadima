# Label Studio

## Обзор
- **URL:** https://labelstud.io
- **Тип:** Open source (Apache 2.0) + Commercial
- **Цена:** Free (self-hosted) / Enterprise
- **Категория:** Data annotation platform

## Описание
Multi-type data annotation: text, images, audio, video. NER, classification, relation extraction. ML backend integration. Enterprise features.

## Сильные стороны
- Multi-modal annotation
- Customizable labeling interface
- ML backend integration (active learning)
- Collaborative
- Enterprise features (SSO, audit)
- REST API
- Docker deployment
- Active development

## Слабые стороны
- No NLP processing (annotation only)
- No term extraction
- No corpus management
- No Hebrew-specific features
- No association measures
- Enterprise features require payment

## Уроки для нашего продукта
ML backend integration for semi-supervised annotation. Enterprise deployment. API-first architecture. Multi-modal approach.
