# INCEpTION

## Обзор
- **URL:** https://inception-project.github.io
- **Тип:** Open source (Apache 2.0)
- **Цена:** Free
- **Категория:** Semantic annotation platform

## Описание
Web-based annotation platform with intelligent assistance. Supports NER, coreference, relation extraction, knowledge linking. Active learning integration.

## Сильные стороны
- Intelligent annotation assistance (ML suggestions)
- Knowledge base integration
- Active learning
- Multiple annotation layers
- Recommendation system
- Export to UIMA, CoNLL, WebAnno
- Customizable
- Docker deployment

## Слабые стороны
- No term extraction
- No corpus management
- No Hebrew-specific features
- Complex setup
- No association measures
- No TM integration
- Steep learning curve

## Уроки для нашего продукта
ML-assisted annotation. Knowledge base linking. Active learning integration. Multiple annotation layers.
