# AntConc

## Обзор
- **URL:** https://www.laurenceanthony.net/software/antconc
- **Тип:** Freeware
- **Цена:** Free
- **Категория:** Corpus analysis

## Описание
Desktop corpus analysis tool. Concordance, word frequency, collocations, n-grams, keywords, clusters. Used by linguists, teachers, translators worldwide.

## Сильные стороны
- Concordance (KWIC) view
- Word frequency lists
- Collocation analysis
- N-gram extraction
- Keyword analysis (vs reference corpus)
- Cluster analysis
- Simple, intuitive UI
- Cross-platform (Java)
- Free

## Слабые стороны
- No morphological analysis
- No POS tagging
- No term extraction (only n-grams)
- No Hebrew-specific features
- No validation framework
- No API
- Desktop only
- No TM integration

## Уроки для нашего продукта
Concordance view. Keyword analysis (corpus vs reference). Collocation statistics. Simple, accessible UI.
