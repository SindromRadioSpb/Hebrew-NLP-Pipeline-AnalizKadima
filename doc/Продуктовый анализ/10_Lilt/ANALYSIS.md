# Lilt

## Обзор
- **URL:** https://lilt.com
- **Тип:** Commercial
- **Цена:** Enterprise pricing
- **Категория:** AI-powered translation platform

## Описание
Enterprise translation platform with adaptive MT. Context-based translation memory. Quality estimation. Adaptive learning from corrections.

## Сильные стороны
- Adaptive MT (learns from corrections)
- Context-aware TM
- Quality estimation
- Enterprise features
- API
- Translation workflow management

## Слабые стороны
- No standalone NLP pipeline
- No term extraction
- No corpus management
- No validation framework for NLP
- Translation-focused, not NLP research
- Expensive

## Уроки для нашего продукта
Adaptive learning from human corrections. Quality estimation. Context-aware matching. Enterprise workflow.
