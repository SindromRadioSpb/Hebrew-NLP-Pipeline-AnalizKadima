# Stanford CoreNLP

## Обзор
- **URL:** https://stanfordnlp.github.io/CoreNLP
- **Тип:** Open source (GPL v3)
- **Цена:** Free
- **Категория:** NLP pipeline

## Описание
Comprehensive NLP pipeline: tokenization, POS, NER, parsing, coreference, sentiment. Supports Hebrew via Stanford NLP models. Server mode for integration.

## Сильные стороны
- Complete pipeline (tokenize → parse → NER → coref)
- Hebrew support (HebPipe integration)
- Constituency and dependency parsing
- Coreference resolution
- Server mode (REST API)
- Well-documented
- Academic backing

## Слабые стороны
- Java-based, heavy
- Slow for large corpora
- No term extraction
- No corpus management
- No validation framework
- Complex setup
- No Hebrew-specific term extraction

## Уроки для нашего продукта
Complete pipeline coverage. Hebrew tokenization rules (HebPipe). Server API architecture. Academic rigor.
