# brat

## Обзор
- **URL:** https://brat.nlplab.org
- **Тип:** Open source (MIT-like)
- **Цена:** Free
- **Категория:** Annotation tool

## Описание
Browser-based rapid annotation tool. Supports NER, relation, event annotation. Collaborative. Visual. Integrates with external resources (Wikipedia).

## Сильные стороны
- Visual annotation interface
- NER + relations + events
- Collaborative (multi-user)
- Wikipedia integration
- Annotation comparison
- Customizable annotation schema
- Web-based, no install

## Слабые стороны
- No NLP processing (annotation only)
- No term extraction
- No corpus management
- No validation metrics
- Dated UI
- No Hebrew-specific features
- No association measures

## Уроки для нашего продукта
Visual annotation workflow. Multi-user collaboration. Annotation comparison. Schema-driven annotation types.
