# Sketch Engine

## Обзор
- **URL:** https://www.sketchengine.eu
- **Тип:** Commercial SaaS
- **Цена:** €99–€499/year
- **Категория:** Corpus management + Term extraction

## Описание
Industry-standard corpus query and term extraction platform. Supports 100+ languages including Hebrew. Word Sketches, Thesaurus, Wordlist, Term Extraction, Bilingual Term Extraction. Used by lexicographers, translators, terminologists.

## Сильные стороны
- Powerful corpus query language (CQL)
- Automatic term extraction with linguistic patterns
- Bilingual term extraction
- Word Sketches — collocation profiles
- 100+ pre-loaded corpora
- Web-based, no installation
- CQL regex for morphologically rich languages
- Export to TBX, CSV, Excel

## Слабые стороны
- No Hebrew-specific morphology pipeline
- No validation/gold corpus framework
- No extraction profile comparison (precise/balanced/recall)
- No TM integration
- No association measure thresholds configuration
- Expensive for individual researchers
- No local/offline deployment option
- Limited API for custom workflows

## Уроки для нашего продукта
Word Sketches as collocation profiles. Term extraction with linguistic patterns + statistics. Corpus query language. Export formats.
