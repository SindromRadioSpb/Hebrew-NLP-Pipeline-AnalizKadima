# TermSuite

## Обзор
- **URL:** https://termsuite.github.io
- **Тип:** Open source (Apache 2.0)
- **Цена:** Free
- **Категория:** Terminology extraction

## Описание
Java-based toolbox for terminology extraction and multilingual term alignment. Morphosyntactic analysis, term variant detection, term specificity computation. Supports French, English, German, and extensible.

## Сильные стороны
- Multiword and compound term detection
- Term variant detection (morphological, syntactic)
- Term specificity computation
- Multilingual term alignment
- Pipeline architecture — extensible
- Open source
- Pattern-based + statistical hybrid approach

## Слабые стороны
- No Hebrew support out of box
- No UI — CLI/library only
- No validation framework
- No corpus management
- No TM projection
- Limited association measures
- No annotation tools
- Documentation gaps

## Уроки для нашего продукта
Term variant detection methodology. Specificity scores. Pattern-based + statistical hybrid. Pipeline architecture for extensibility.
