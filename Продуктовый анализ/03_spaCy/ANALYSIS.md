# spaCy

## Обзор
- **URL:** https://spacy.io
- **Тип:** Open source (MIT)
- **Цена:** Free
- **Категория:** NLP framework

## Описание
Industrial-strength NLP library. Tokenization, POS tagging, NER, dependency parsing, sentence segmentation. Supports 25+ languages. Production-ready, fast, Python.

## Сильные стороны
- Fast, production-ready
- Modular pipeline architecture
- Custom components support
- Language models with trained pipelines
- Dependency parsing
- NER
- Active community
- Integration with transformers (spacy-transformers)

## Слабые стороны
- No Hebrew model out of box (community models exist)
- No term extraction built-in
- No corpus management
- No validation framework
- No association measures
- No TM integration
- No GUI

## Уроки для нашего продукта
Pipeline architecture (component-based). Custom component extensibility. Fast tokenization. Language model integration.
